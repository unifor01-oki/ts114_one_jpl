# 이미지 파일 안내

다음 이미지 파일들이 이 폴더에 있습니다:

1. **card1-rings.jpg** - 오프라인 업무 카드용 이미지 (반지들이 진열된 사진)
2. **card2-laptop.jpg** - 온라인 업무 카드용 이미지 (노트북 대시보드 사진)
3. **card3-cloud.jpg** - 커넥트 드라이브 카드용 이미지 (클라우드 저장소 아이콘들)

## 사용 방법

- 위 파일명으로 이미지를 넣으면 자동으로 카드에 표시됩니다.
- 이미지가 없으면 빈 박스(점선 테두리)가 표시됩니다.
- 지원 형식: JPG, PNG, WEBP 등 일반적인 이미지 형식
